This project is a pixel-perfect clone of the **Netflix home page UI**, built with modern frontend technologies to replicate the look and feel of Netflix’s main screen.

- Built using **React, Next.js, Tailwind CSS**
- Fully responsive design for desktop and mobile
- Dynamic content sections for featured movies and categories

## Install

```sh-session
git clone https://github.com/Sujal-py3/Netflix-Clone.git
cd Netflix-Clone
npm install
